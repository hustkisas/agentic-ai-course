# Weekly review + plan next week

!!! success "Weekly ritual (every Sunday)"
    Scheduled: **2026-07-19** · Week 2 · Phase 1. 30 minutes. The habit that keeps the plan alive.

## The 5-question review

1. **What did I actually finish this week vs planned?** (Prompting lesson + exercise? Tool-calling lesson + one-cycle exercise? Read the article? Milestone project?)
2. **What slipped, and why?** Cause, not guilt.
3. **Reschedule the slippage now** into specific slots next week.
4. **Did I build, or just read?** The tool-cycle project is the proof of Phase 1 — did you run it?
5. **Are next week's calendar blocks set?** Week 3 starts Phase 2 (the agent loop) — protect the time; this is the heart of the course.

## Week 2 self-check

You're ready for Phase 2 when you can:

- [ ] Explain the three message roles and use a system prompt + few-shot to control output format.
- [ ] Get reliable structured (JSON) output and parse it in Python.
- [ ] Explain the tool-calling mechanic: model *requests*, your code *executes*, you feed the result back.
- [ ] State the workflow-vs-agent distinction and when to use each (from "Building Effective Agents").
- [ ] **Run your milestone project**: one full tool cycle (declare → request → execute → feed back → final answer).

If any box is unchecked, that's your first task next week — before starting the agent loop.

## Looking ahead: Phase 2 (Weeks 3–6)

This is the core of the whole course. You'll combine what you now know — the loop (Week 1) and tools (Week 2) — into an actual agent you build from scratch, then add memory and RAG. It's the most important phase; arrive with Phase 1 solid.
