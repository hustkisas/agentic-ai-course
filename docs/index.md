# Agentic AI — Self-Study Course

An 11-week, hands-on path from LLM fundamentals to a deployed agent. Built for a systems/algorithms programmer moving into agentic AI. **Python track.**

[Start here → The Agent Loop lesson](week03/01-the-agent-loop.md){ .md-button .md-button--primary }

## How this works

- **7 phases / 11 weeks / 55 sessions.** Use the left nav (or search) to jump around.
- Each session has a scheduled date, an objective, and — where relevant — the exact Pi repo file to study.
- **Milestone projects** are the checkpoints that prove you can build, not just read.
- Full written lessons appear as you reach them (the Agent Loop lesson is done as your flagship example).

## The 7 phases

| Phase | Weeks | Focus |
|---|---|---|
| 0 · Foundations | 1 | async Python, HTTP/SSE, LLM mental model |
| 1 · LLM fundamentals | 2 | prompting, tool calling |
| 2 · Agent loop & core methods | 3–6 | build an agent from scratch, then RAG |
| 3 · Orchestration & multi-agent | 7–8 | frameworks, MCP, supervisors |
| 4 · Harness engineering | 9 | sessions, streaming, providers (study Pi) |
| 5 · Eval & observability | 10 | evals, tracing |
| 6 · Deployment & capstone | 11 | FastAPI, Docker, ship it |

## Companion trackers
